This project was made for a univeristy software engineering class

## How to run

### On Windows (Command Prompt/PowerShell): 
./mvnw spring-boot:run

### On Mac / Linux: 
chmod +x mvnw   (Only needed once to grant execution permission)
./mvnw spring-boot:run
